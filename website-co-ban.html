Nhập Môn HTML/CSS, Javascript

1. Giới thiệu chung

- HTML được ví như là khung xương của 1 cô gái, CSS sẽ makeup cho cô gái thêm xinh đẹp, Javascript trang bị cho cô gái những cử chỉ hành động.

- HTML là ngôn ngữ đánh dấu siêu văn bản, được tổ chức theo các thẻ, kết hợp với CSS và Javascript để thiết kế Website.

- HTML được viết trong file có đuôi dạng *.html hoặc *.htm
ví dụ: index.html, index.htm

- CSS được viết trong file có đuôi dạng *.css
ví dụ: style.css

- Javascript được viết trong file có đuôi dạng *.js
ví dụ: custom.js

- Những file này được chạy khi mở trên các trình duyệt như:

Google Chrome Mozila Firefox Google Chromium
Internet Explorer Microsoft Edge CốcCốc

- Để tạo và chỉnh sửa chúng, ta thường dùng các phần mềm chỉnh sửa text( text editor) hoặc phần mềm phát triển ứng dụng (IDE).

- Một số text editor và IDE phổ biến hiện nay:

Visual studio Sublime text Atom
Notepad++ Brackets IDE Inteliji

Để được trực quan hơn, dưới đây là 1 ví dụ về HTML, CSS, JS

Dưới đây là 1 ví dụ về HTML

Hãy điền dòng text này vào phần HTML: <p>Hello</p>

Sau đó điền dòng text này vào phần CSS: p{color:red;}

Điền dòng text này vào phần Javascript: alert("hello"); trình duyệt sẽ đưa ra thông báo.
